# 주식 투자 에이전트 - 필요 데이터 및 수집처

## 분석 순서와 데이터의 관계

투자 판단은 큰 그림에서 작은 그림 순서로 좁혀가야 한다. 차트만 보거나 뉴스만 보면 함정에 빠지기 쉽다. **시장 국면이 나쁠 때는 차트가 아무리 좋아도 가짜 신호일 수 있기 때문이다.**

```
1단계: 시장 국면 파악   → 매크로 지표, 지수 데이터
       "지금 상승장인가, 하락장인가, 횡보인가?"

2단계: 섹터 흐름 파악   → 섹터별 수급, 외인/기관 동향
       "어느 섹터에 돈이 몰리고 있나?"

3단계: 종목 차트 분석   → OHLCV, 기술적 지표, 차트 패턴
       "시장 흐름에 올라탄 종목 중 차트가 좋은 것은?"

4단계: 크로스체크       → 재무제표, 공시, 뉴스
       "차트가 좋은 종목이 실제로 괜찮은 회사인가?"
```

각 데이터 카테고리는 이 4단계 중 하나 이상에 대응된다.

---

## 데이터 카테고리 1: 주가 & 차트 데이터 (3단계 대응)

### 왜 필요한가?
- 언제 사고 팔지 판단하는 기본 원재료
- 기술적 지표 및 차트 패턴 계산의 기반
- 백테스트의 핵심 데이터

### 필요한 구체적 항목
- 일별 시가/고가/저가/종가/거래량 (OHLCV)
- 시가총액 (대형주/소형주 구분용)
- 외국인·기관 매매 동향 (큰 손의 움직임 파악)

### 차트 패턴 & 기술적 지표 (OHLCV에서 계산)
OHLCV 원본 데이터를 바탕으로 아래 항목을 자동 계산한다. 별도 수집 불필요.

| 유형 | 항목 | 용도 |
|-----|------|------|
| 추세 지표 | 이동평균(5/20/60/120일), MACD | 추세 방향 확인 |
| 모멘텀 지표 | RSI, 스토캐스틱 | 과매수/과매도 구간 판단 |
| 변동성 지표 | 볼린저밴드, ATR | 변동성 구간 및 돌파 감지 |
| 거래량 지표 | 거래량 이동평균, OBV | 수급 확인 |
| 차트 패턴 | 골든/데드크로스, 이중바닥, 지지/저항선 | 매매 시그널 |

### 어디서 구하나?

| 출처 | 제공 데이터 | 비용 | 접근 방법 |
|-----|-----------|------|---------|
| **KRX (한국거래소)** | 공식 주가 데이터 | 무료 | `pykrx` 라이브러리 |
| **한국투자증권 Open API** | 실시간 시세, 호가 | 무료 (계좌 필요) | REST API |
| **네이버 금융** | 주가, 차트 | 무료 (비공식) | 크롤링 |
| **FinanceDataReader** | 국내외 주가 통합 | 무료 | Python 라이브러리 |

**선택:** `pykrx` + `한국투자증권 Open API` 조합
- pykrx: 과거 데이터 수집 (무료, 안정적)
- KIS API: 실시간 시세 + 실제 주문 실행

## 데이터 카테고리 2: 재무제표 데이터

### 왜 필요한가?
- "좋은 회사"를 구별하는 핵심 근거
- 단순히 주가가 올랐다고 좋은 게 아니라, 실적이 뒷받침되는지 확인 필요
- PER, PBR 같은 밸류에이션 지표 계산의 기반

### 필요한 구체적 항목
- **손익계산서**: 매출, 영업이익, 순이익 (회사가 돈을 버는가?)
- **대차대조표**: 자산, 부채, 자본 (회사가 재무적으로 안전한가?)
- **현금흐름표**: 실제 현금이 들어오는가? (이익이 현금화되는가?)
- **분기별 실적**: 트렌드 파악 (성장하는가, 둔화되는가?)

### 어디서 구하나?

| 출처 | 제공 데이터 | 비용 | 접근 방법 |
|-----|-----------|------|---------|
| **DART (금융감독원)** | 전체 상장사 공식 재무제표 | 무료 | DART Open API (API 키 필요) |
| **FnGuide** | 정제된 재무 데이터 | 유료 | 계약 필요 |
| **네이버 금융** | 요약 재무 데이터 | 무료 (비공식) | 크롤링 |
| **KIS API** | 기본 재무 지표 | 무료 (계좌 필요) | REST API |

**선택:** DART Open API (공식, 무료, 신뢰성 높음)
- `dart-fss` 또는 `OpenDartReader` Python 라이브러리 활용
- API 키 발급: https://opendart.fss.or.kr (무료)

## 데이터 카테고리 3: 공시 데이터

### 왜 필요한가?
- 회사의 중요한 변화를 가장 먼저 공개적으로 알 수 있는 채널
- 유상증자(주식 가치 희석), 합병, 대규모 계약 등 주가에 큰 영향
- 주요 주주나 임원의 지분 변동으로 내부자 동향 파악 가능

### 필요한 구체적 항목
- 사업보고서, 반기보고서, 분기보고서
- 주요 경영사항 공시 (계약, 투자, M&A 등)
- 지분 공시 (대주주 지분 변동)
- 유상/무상증자, 자사주 매입

### 어디서 구하나?

| 출처 | 제공 데이터 | 비용 | 접근 방법 |
|-----|-----------|------|---------|
| **DART (금융감독원)** | 전체 상장사 모든 공시 | 무료 | DART Open API |

**선택:** DART Open API (사실상 유일한 공식 무료 출처)
- 재무제표와 동일한 API 키로 접근 가능
- 공시 발생 시 실시간(30분 이내) 수집 가능

## 데이터 카테고리 4: 뉴스 & 감성 데이터

### 왜 필요한가?
- 주가는 종종 재무제표보다 뉴스에 먼저 반응
- 부정적 이슈(사고, 소송, 실적 쇼크)를 조기에 감지해 손실 방어
- 긍정적 뉴스(대형 계약, 신제품)로 매수 기회 포착

### 필요한 구체적 항목
- 종목별 뉴스 (회사 이름으로 필터링)
- 시황 뉴스 (오늘 시장 전반적 분위기)
- 산업/섹터 뉴스 (반도체, 2차전지 등 주요 산업 동향)
- 애널리스트 리포트 요약 (목표가, 투자의견 변경)

### 어디서 구하나?

| 출처 | 제공 데이터 | 비용 | 접근 방법 |
|-----|-----------|------|---------|
| **네이버 금융 뉴스** | 종목별 뉴스, 시황 | 무료 (비공식) | 크롤링 또는 RSS |
| **한국경제, 머니투데이 등** | 경제 뉴스 | 무료 | RSS 피드 |
| **BigKinds (한국언론재단)** | 정제된 뉴스 검색 | 일부 무료 | API |

**선택:** 네이버 금융 RSS + 크롤링 (무료, 커버리지 넓음)
- 주의: 크롤링은 서비스 약관 확인 필요, 과도한 요청 금지
- 장기적으로 BigKinds API 도입 검토

## 데이터 카테고리 5: 매크로 경제 데이터

### 왜 필요한가?
- 주식 시장 전체의 방향성은 금리, 환율 같은 거시 지표에 크게 영향받음
- 금리 인상기에는 성장주 조심, 금리 인하기에는 공격적 투자 가능
- 환율이 오르면 수출 기업 유리, 달러 강세면 외국인 자금 유출 우려

### 필요한 구체적 항목
- **기준금리**: 한국은행 기준금리 결정 방향
- **시장 금리**: 국고채 3년/10년 금리
- **환율**: 원/달러 환율
- **인플레이션**: 소비자물가지수(CPI)
- **경기 지표**: GDP 성장률, 산업생산지수

### 어디서 구하나?

| 출처 | 제공 데이터 | 비용 | 접근 방법 |
|-----|-----------|------|---------|
| **한국은행 ECOS** | 모든 국내 매크로 지표 | 무료 | ECOS Open API (API 키 필요) |
| **통계청 KOSIS** | 경제통계 | 무료 | KOSIS Open API |
| **pykrx** | KOSPI/KOSDAQ 지수 | 무료 | Python 라이브러리 |

**선택:** 한국은행 ECOS API (공식, 무료, 가장 신뢰성 높음)
- API 키 발급: https://ecos.bok.or.kr (무료)

## 데이터 수집 가능 여부 요약

| 카테고리 | 무료 수집 가능? | 핵심 출처 | 난이도 |
|---------|-------------|---------|-------|
| 주가 (과거) | ✅ 가능 | pykrx | 쉬움 |
| 주가 (실시간) | ✅ 가능 (계좌 필요) | KIS API | 보통 |
| 재무제표 | ✅ 가능 | DART API | 보통 |
| 공시 | ✅ 가능 | DART API | 보통 |
| 뉴스 | ✅ 가능 (비공식) | 네이버 크롤링 | 보통 |
| 매크로 | ✅ 가능 | 한국은행 ECOS | 쉬움 |
| 실시간 호가/체결 | ⚠️ 유료 또는 계좌 필요 | KIS API | 어려움 |
| 증권사 리포트 원문 | ❌ 어려움 | 없음 | - |

**결론: 핵심 데이터는 모두 무료로 수집 가능**

## 발급 받아야 할 API 키 목록

아래 순서대로 미리 발급해두는 것을 권장:

1. **DART Open API** (재무제표 + 공시)
   - 발급: https://opendart.fss.or.kr/uat/uia/egovLoginUsr.do
   - 소요 시간: 즉시 발급
   - 일일 한도: 10,000건

2. **한국투자증권 Open API** (주가 실시간 + 주문)
   - 발급: https://apiportal.koreainvestment.com
   - 소요 시간: 계좌 개설 후 신청 (1~2일)
   - 모의투자 계좌 먼저 개설 권장

3. **한국은행 ECOS API** (매크로 지표)
   - 발급: https://ecos.bok.or.kr/#/AuthKeyApply
   - 소요 시간: 즉시 발급
   - 일일 한도: 1,000건

4. **Claude API** (AI 분석 + 전략 수립)
   - 발급: https://console.anthropic.com
   - 소요 시간: 즉시 (카드 등록 필요)
   - 비용: 사용량에 따라 과금

5. **Telegram Bot** (알림)
   - 발급: Telegram에서 @BotFather 검색
   - 소요 시간: 5분

## 데이터 접근 시 주의사항

- **크롤링**: 네이버, 다음 등 포털 크롤링은 robots.txt 및 이용약관 확인 필수. 과도한 요청은 IP 차단 위험
- **개인정보**: 수집 데이터에 개인정보 없음 (모두 공개 시장 데이터)
- **저작권**: 공시 원문, 뉴스 본문은 저작권 있음 → 분석 목적 내부 사용만 허용
- **API 한도**: DART, ECOS 등은 일일 호출 한도 있음 → 불필요한 중복 호출 방지 필요
