# 주식 투자 에이전트 - 분석 엔진 단계별 계획

## 개요

분석 엔진은 큰 그림 → 작은 그림 순서로 4단계를 거쳐 최종 매매 후보를 좁혀간다.
앞 단계의 판단이 뒷 단계의 해석 기준에 영향을 준다.

```
전체 KOSPI 200 종목
        │
[1단계] 시장 국면 파악
        │ 국면에 따라 이후 분석 기준 강도 조정
        │
[2단계] 섹터 흐름 파악
        │ 주도 섹터 종목만 통과
        │
[3단계] 종목 차트 분석
        │ 차트 점수 상위 종목만 통과
        │
[4단계] 재무 + 뉴스 크로스체크
        │
최종 매수 후보 (3~10개 종목)
```

## 파라미터 출처 표기 방식

각 판단 기준 옆에 출처를 표기한다:
- ✅ **검증됨**: 학술 연구 또는 널리 검증된 실무 기준
- 🔶 **통상 관행**: 시장 참여자들이 관행적으로 사용하지만 절대적 기준은 아님
- 🧪 **미검증**: 합리적으로 설계했으나 한국 시장 백테스트로 검증 필요

---

## 1단계: 시장 국면 파악

**목적:** 지금이 공격적으로 매수해도 되는 장인지, 아니면 현금 보유를 늘려야 할지 판단

### 사용 데이터

| 데이터 | 출처 | 구체적 항목 |
|-------|------|-----------|
| 지수 | pykrx | KOSPI, KOSDAQ 일별 종가 및 거래대금 |
| 지수 기술적 지표 | 계산값 | KOSPI 60일/200일 이동평균 대비 현재 위치 |
| 외국인 수급 | pykrx | 외국인 KOSPI 순매수 5일 누적 |
| 금리 | 한국은행 ECOS | 국고채 3년 금리, 기준금리 |
| 환율 | 한국은행 ECOS | 원/달러 환율 |

### 판단 로직

```
시장 국면 점수 계산:

  ✅ (+2) KOSPI가 200일 이동평균 위에 있음
         → Meb Faber "Global Tactical Asset Allocation" (2007) 연구
           200일 이동평균 위이면 매수, 아래이면 현금 보유가 유효함을 검증

  🔶 (+1) 외국인이 최근 5일 순매수
         → 한국 시장 특성상 외국인 수급이 지수 방향과 강하게 연동됨
           (수급이 추세를 만든다는 관행적 믿음, 정량 연구는 제한적)

  🔶 (+1) 원/달러 환율 안정 (전주 대비 ±1% 이내)
         → 환율 급등 시 외국인 자금 이탈 경향. 임계값(1%)은 🧪 검증 필요

  🔶 (-1) 국고채 3년 금리 급등 중 (전월 대비 +0.3%p 이상)
         → 금리 상승 = 할인율 상승 = 주식 가치 하락. 임계값은 🧪 검증 필요

  ✅ (-2) KOSPI가 200일 이동평균 아래에 있음
         → 위와 동일 (Faber 연구)

  🔶 (-1) 외국인이 최근 5일 순매도

점수 → 국면 판정:
  🧪 3점 이상  : 상승장 → 공격적 매수 허용   ← 임계값 백테스트 필요
  🧪 1~2점     : 중립장 → 선별적 매수
  🧪 0점 이하  : 하락장 → 매수 중단, 현금 보유
```

### 1단계 출력

```json
{
  "market_phase": "uptrend",
  "phase_score": 3,
  "buy_aggressiveness": "high",
  "summary": "KOSPI가 200일선 위에 위치하고 외국인 5일 연속 순매수. 공격적 매수 가능."
}
```

## 2단계: 섹터 흐름 파악

**목적:** 어느 섹터로 돈이 몰리는지 파악해 시장의 주도 테마에 올라탐

### 사용 데이터

| 데이터 | 출처 | 구체적 항목 |
|-------|------|-----------|
| 섹터별 수익률 | pykrx | GICS 기준 섹터 최근 5일/20일 수익률 |
| 섹터별 외국인 수급 | pykrx | 섹터별 외국인 순매수 금액 |
| 섹터별 기관 수급 | pykrx | 섹터별 기관 순매수 금액 |
| 섹터별 거래대금 | pykrx | 섹터별 거래대금 변화율 |

### 판단 로직

```
섹터별 점수 산정:

  ✅ 5일/20일 수익률 상위 섹터 선별
     → Jegadeesh & Titman (1993) 모멘텀 연구: 최근 수익률 높은 자산이
       단기적으로 계속 높은 수익률을 보이는 경향 (모멘텀 효과)
       원래는 개별 종목 기준이나 섹터에도 적용됨

  🔶 외국인/기관 순매수 섹터 우선
     → 한국 시장에서 외국인·기관 수급이 섹터 주도권과 연동되는
       관행적 기준. 정량 연구보다는 실무 경험에 기반

  🧪 거래대금 150% 이상 급증 섹터 주목
     → 거래량 급증 = 시장의 관심 집중으로 해석. 임계값(150%)은 검증 필요

결과:
  🧪 주도 섹터: 점수 합 5점 이상   ← 임계값 백테스트 필요
  🧪 관심 섹터: 점수 합 3~4점
  🧪 소외 섹터: 점수 합 2점 이하 → 해당 섹터 종목 제외
```

### 2단계 출력

```json
{
  "leading_sectors": ["반도체", "전기전자"],
  "watch_sectors": ["2차전지"],
  "avoid_sectors": ["건설", "금융", "유통"],
  "summary": "반도체 섹터 외국인 5일 연속 순매수, 섹터 수익률 상위. 건설·금융 소외 중."
}
```

## 3단계: 종목 차트 분석

**목적:** 주도 섹터 내에서 실제로 매수 타이밍이 온 종목 선별

**입력:** 2단계에서 선정된 주도/관심 섹터 종목들만 분석

### 사용 데이터

| 데이터 | 출처 | 구체적 항목 |
|-------|------|-----------|
| 일별 OHLCV | pykrx | 최근 120일치 시가/고가/저가/종가/거래량 |
| 기술적 지표 | 계산값 (pandas-ta) | 아래 항목 |

### 계산하는 기술적 지표

| 지표 | 파라미터 | 출처 | 사용 목적 |
|-----|---------|------|---------|
| 이동평균 | 5/20/60일 SMA | 🔶 국내 시장 관행 (20/60일이 단기/중기 기준으로 통용) | 추세 방향 확인 |
| 골든/데드크로스 | 5일선 vs 20일선 | 🔶 실무 관행. 원래는 50/200일선이 정통 기준 | 추세 전환 시그널 |
| RSI | 14일 | ✅ J. Welles Wilder "New Concepts in Technical Trading Systems" (1978) 원저자 제시 파라미터 | 과매수/과매도 판단 |
| RSI 기준선 | 70/30 | ✅ Wilder 원저 기준. 단, 🔶 상승장에서는 80/40 적용 사례도 많음 | |
| MACD | 12/26/9 | ✅ Gerald Appel이 제시한 원래 파라미터. 전 세계 표준으로 사용 | 모멘텀 전환 |
| 볼린저밴드 | 20일, ±2σ | ✅ John Bollinger "Bollinger on Bollinger Bands" (2001) 원저자 제시 파라미터 | 가격 위치, 변동성 |
| 거래량 비율 | 20일 평균 대비 | 🔶 거래량 급증을 수급 확인 지표로 쓰는 실무 관행 | 수급 확인 |
| 지지/저항 | 최근 60일 고점/저점 | 🔶 실무 관행. 기간 설정은 🧪 검증 필요 | 돌파 여부 판단 |

### 매수 시그널 패턴

| 패턴명 | 조건 | 출처 |
|-------|------|------|
| 눌림목 매수 | 20일선 근처 거래량 감소 후 양봉 | 🔶 국내 단기 트레이더들이 많이 사용하는 패턴 |
| 골든크로스 | 단기선이 장기선 상향 돌파 + 거래량 증가 | 🔶 실무 관행. 후행성 있어 단독 사용 시 신뢰도 낮음 |
| 볼린저 수축 돌파 | 밴드 폭 수축(Squeeze) 후 상단 돌파 + 거래량 2배↑ | ✅ John Bollinger가 제시한 Squeeze 전략 |
| RSI 반등 | RSI 30~40 구간에서 상승 전환 | ✅ Wilder RSI 기준 응용 |
| 신고가 돌파 | 60일 고점 상향 돌파 + 거래량 급증 | 🔶 William O'Neil "How to Make Money in Stocks" CANSLIM 전략의 신고가 돌파 원칙 |

### 차트 점수 산정

```
🧪 아래 점수 체계 전체가 백테스트 검증 필요 (임계값, 가중치 모두)

  (+3) 명확한 매수 패턴 1개 이상 감지
  (+2) 20일 이동평균 위에 위치
  (+2) 거래량 비율 1.5배 이상
  (+1) RSI 40~60 (적정 구간)
  (-2) RSI 70 이상 (과매수)
  (-3) 60일 이동평균 아래에 위치
  (-1) 거래량 감소 추세

7점 이상: 매수 적합 → 4단계로 전달   🧪
4~6점: 관찰 대상
3점 이하: 제외
```

  "name": "삼성전자",
  "chart_score": 8,

  "rsi": 52,
  "position_vs_ma20": "above",
  "summary": "볼린저밴드 수축 후 상단 돌파, 거래량 평소 2.3배. 매수 타이밍 적합."
}

## 4단계: 재무 + 뉴스 크로스체크

**목적:** 차트가 좋은 종목이 실제로 투자해도 되는 회사인지 최종 확인

**입력:** 3단계에서 차트 점수 7점 이상인 종목들

### 사용 데이터

| 데이터 | 출처 | 구체적 항목 |
|-------|------|-----------|
| 밸류에이션 | pykrx / DART | PER, PBR, 업종 평균 대비 수준 |
| 수익성 | DART | 최근 4분기 영업이익, 순이익 추이 |
| 재무 안전성 | DART | 부채비율, 유동비율 |
| 성장성 | DART | 매출/영업이익 전년동기 대비 성장률 |
| 공시 | DART | 최근 30일 주요 공시 |
| 뉴스 | 크롤링 | 최근 7일 종목 관련 뉴스 감성 점수 |

### 판단 로직

**즉시 제외 조건**

```
✅ 최근 분기 영업이익 적자 전환
   → 피오트로스키 F-Score (Piotroski, 2000): 수익성 악화는 핵심 탈락 기준

✅ 부채비율 200% 초과
   → 재무 안전성 기준으로 국내외 퀀트 전략에서 공통 사용
     (업종별 차이 있음 — 금융업은 별도 기준 적용 필요)

✅ 유상증자 공시
   → 주식 수 증가 = EPS 희석. 단기 주가 하락 경향 실증 연구 다수

✅ 대주주 지분 대량 매도 공시
   → 내부자 매도는 부정적 신호. Seyhun (1986) 내부자 거래 연구

🔶 뉴스 감성 점수 -0.5 이하 (강한 부정)
   → 감성 분석 임계값은 🧪 검증 필요

✅ 불성실 공시 법인 지정
   → 공시 신뢰도 훼손, 기본 제외 기준
```

**가산/감산 점수**

```
✅ (+2) 영업이익 전년동기 대비 20% 이상 성장
        → CANSLIM의 'C'(Current Earnings) 기준 응용
          O'Neil은 최소 25% 성장을 기준으로 제시. 🧪 한국 시장 임계값 검증 필요

✅ (+1) PER이 업종 평균보다 낮음 (저평가)
        → 가치투자 기본 원칙. Fama & French (1992) 가치주 효과 연구

🔶 (+1) 외국인 지분율 최근 1달 증가
        → 한국 시장 특성. 외국인 매집 = 긍정 신호로 보는 실무 관행

🔶 (+1) 최근 7일 뉴스 감성 긍정적
        → 감성 분석의 주가 예측력에 대한 연구 있으나 🧪 임계값 검증 필요

🧪 (-1) PER이 업종 평균의 1.5배 이상 (고평가) ← 임계값 검증 필요

✅ (-1) 영업이익 성장 둔화 (2분기 연속 감소)
        → 피오트로스키 F-Score의 수익성 변화 지표

🔶 (-2) 최근 뉴스 부정적 이슈 (소송, 사고 등)
```

**재무 점수 체계 (피오트로스키 F-Score 참고)**

피오트로스키 F-Score는 9개 항목으로 재무 건전성을 0~9점으로 평가하는 검증된 방법론이다 (Piotroski, 2000, Journal of Accounting Research). 우리 시스템은 이를 간소화해 적용한다.

| 항목 | 기준 | 점수 | 원래 F-Score 항목 |
|-----|------|------|-----------------|
| 수익성 | 영업이익 > 0 | +1 | ROA > 0 |
| 수익성 개선 | 영업이익 전년 대비 증가 | +1 | ΔROA > 0 |
| 현금흐름 | 영업현금흐름 > 0 | +1 | CFO > 0 |
| 재무 안전성 | 부채비율 감소 | +1 | ΔLeverage < 0 |
| 성장성 | 매출 전년 대비 증가 | +1 | ΔMargin > 0 |

> 5점 만점 중 4점 이상이면 재무 건전 판정. 🧪 임계값 백테스트 필요

### 4단계 출력

```json
{
  "ticker": "005930",
  "name": "삼성전자",
  "fundamental_score": 4,
  "instant_exclude": false,
  "per": 12.4,
  "sector_avg_per": 15.2,
  "operating_profit_growth": "+18%",
  "news_sentiment": 0.3,
  "recent_disclosures": ["자사주 매입 결정"],
  "final_verdict": "매수 적합",
  "summary": "PER 업종 평균 대비 저평가, 영업이익 성장세 유지. 자사주 매입 공시 긍정적."
}
```

## 최종 종목 선정

4단계를 통과한 종목을 차트 점수 + 재무 점수 합산으로 순위 정렬.

```
최종 점수 = 차트 점수 × 0.6 + 재무 점수 × 0.4
```

> 🧪 0.6/0.4 가중치는 임의 설정. 백테스트로 최적 비율 탐색 필요.
> 타이밍(차트)과 안전성(재무) 중 어디에 더 비중을 둘지에 따라 조정.

## LLM(Claude)이 개입하는 시점

모든 단계를 코드(규칙 기반)로 처리하되, 아래 시점에 Claude API를 호출해 판단을 보완한다.

| 시점 | LLM 역할 |
|-----|---------|
| 1단계 후 | 시장 국면 점수 + 매크로 뉴스를 종합해 한 줄 시황 해석 |
| 2단계 후 | 섹터 흐름의 이유 해석 (단순 수치가 아닌 맥락 파악) |
| 4단계 후 | 최종 후보 종목들에 대한 종합 의견 및 우선순위 제안 |
| 전략 수립 | 현재 시장 국면에 맞는 투자 전략 전체 설계 |

> **LLM은 판단을 돕는 역할, 최종 실행은 항상 규칙 기반 코드가 담당.**
> LLM 환각(Hallucination)이 직접 매매로 이어지는 것을 방지하기 위함.
